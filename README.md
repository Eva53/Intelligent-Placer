# Intelligent Placer #
## Постановка задачи ##

На вход поступает фотография, на которой имеется некоторое количество предметов. Также на фотографии находится белый лист бумаги А4, на котором черным перманентным маркером нарисован многоугольник произвольного размера.

Необходимо определить, могут ли каким-либо способом данные предметы одновременно, но не перекрывая друг друга, поместиться в этот многоугольник.

## Требования ##

### Общие требования ###

* Программа получает на вход путь до изображения со всеми объектами и нарисованным на листе многоугольником
* Ответом может быть либо True в случае успеха, либо False в случае, если предметы не могут быть умещены в многоугольник, или же изображение, поступившее на вход, не удовлетворяет требованиям
* Ответ выводится в стандартный поток вывода

### Требования к входным данным ###

* Фотографии должны быть в формате *.png
* Фотографии сделаны под прямым углом к нормали поверхности с допущением небольшого отклонения до 10°
* Освещение и цвета объектов должны быть подобраны так, чтобы они были легко отличимы от поверхности
* Следует избегать теней от предметов, для этого можно воспользоваться дополнительным освещением

### Требования к многоугольнику ###

* Многоугольник нарисован на отдельном листе чистой белой бумаги чёрным перманентным маркером
* Многоугольник находится в верхней части изображения
* Многоугольник должен целиком помещаться на фото

### Требования к предметам ###

* Предметы должны быть размещены на белой поверхности
* Предметы не пересекаются друг с другом и не имеют общих границ
* Предметы не сливаются с белым фоном
* Предмет может присутствовать на изображении только в единственном экземпляре
* Предметы не должны пересекаться с многоугольником и должны находиться вне его
* Предметы должны целиком помещаться на фото

### Требования к исходным данным ###

* Исходные данные содержат изображения десяти различных предметов
* Фотографии исходных предметов сделаны на чистом белом листе A4, при этом предметы находятся в центре листа
* Края листов бумаги на фото хорошо видны

# План-алгоритм решения задачи #

1. Преобразовать изображение из цветного в черно-белое
2. Найти на фотографии нарисованный **черным** маркером многоугольник и построить его маску
3. Найти на фотографии объекты по их контрасту с поверхностью, определить их границы и построить маски
4. Проверить выполнение требований для поданного изображения, если хотя бы одно из них не выполнено, программа выдаст False
5. Найти сумму площадей объектов и сравнить с площадью многоугольника. Если суммарная площадь объектов больше - объекты никаким образом не могут быть расположены внутри многоугольника - на выход программа подаст False
6. С помощью перебора, начиная с **большего** по площади объекта и далее по убыванию, по очереди перемещать объекты внутрь многоугольника (Замечание: помещенные во многоугольник предметы будут учитываться при перемещении уже других предметов), возможно с вращением, так чтобы границы объектов лежали строго внутри многоугольника и не пересекались между собой. Если такое перемещение возможно, программа выдаст True, в обратном случае False.
